üîß Installation Guide

Windows Installation:

  1. Download yt-dlp:
    • Visit https://github.com/yt-dlp/yt-dlp/releases
    • Download yt-dlp.exe for Windows
  2. Place the file:
    • Create a folder: C:\yt-dlp
    • Move yt-dlp.exe to this folder
  3. Add to PATH (Optional but Recommended):
    • Right-click "This PC" ‚Üí Properties ‚Üí Advanced System Settings
    • Click "Environment Variables"
    • Under "System variables", find "Path" and click "Edit"
    • Click "New" and add: C:\yt-dlp
    • Click "OK" on all windows
  4. Install FFmpeg (Required for merging video/audio):
    • Download FFmpeg from https://ffmpeg.org/download.html
    • Extract and add FFmpeg's bin folder to PATH (same process as step 3)
Note: After adding to PATH, restart Command Prompt for changes to take effect.

üí° Usage Examples

Example 1: Download Best Quality Video

yt-dlp "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

Example 2: Download MP3 Audio

yt-dlp -x --audio-format mp3 "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

Example 3: Download with Custom Output Location

yt-dlp -o "C:\Music\%(title)s.%(ext)s" "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

‚ö° Quick Tips

  • Update yt-dlp regularly: Run yt-dlp -U to get the latest version
  • Check available formats: Use yt-dlp -F "URL" to see all formats
  • Resume downloads: yt-dlp automatically resumes interrupted downloads
  • Download playlists: Just paste the playlist URL, it works automatically
  • Use quotes: Always wrap URLs in quotes to avoid errors

üõ†Ô∏è Troubleshooting

  • "yt-dlp is not recognized": Either add yt-dlp to PATH or run commands from the folder containing yt-dlp.exe
  • Audio and video not merging: Install FFmpeg and add it to PATH
  • Download fails: Update yt-dlp using yt-dlp -U
  • Slow downloads: Your ISP might be throttling, try using --limit-rate parameter
Pro Tip: You can combine multiple options! For example:
yt-dlp -x --audio-format mp3 --embed-thumbnail --add-metadata -o "%(title)s.%(ext)s" "URL"
This downloads audio as MP3, embeds thumbnail, adds metadata, and uses video title as filename.
